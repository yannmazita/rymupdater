# Petit pendu
Petit jeu du pendu

## Installation
Python >= 3.10

### Poetry
```commandline
poetry install
```
### Sinon
Par exemple dans un venv :
```commandline
pip install -r requirements.txt
```

## Lancement

### Poetry
À la racine du projet :
```commandline
poetry run python -um src.presentation.app 
```
### Sinon
Par exemple dans un venv à la racine du projet :
```commandline
python -um src.presentation.app
```
